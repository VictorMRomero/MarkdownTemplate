# How to Write in Markdown

## Headers

Below is how to write headers in Markdown and their respective results:

### Syntax:
```markdown
# Title
## Subtitle
### Sub-subtitle
#### Level 4 Title
##### Level 5 Title
###### Level 6 Title
```

### Result:
# Title
## Subtitle
### Sub-subtitle
#### Level 4 Title
##### Level 5 Title
###### Level 6 Title
